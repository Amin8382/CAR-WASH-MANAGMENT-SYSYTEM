## Overview

Full-stack web management system for car wash businesses. Provides user authentication, booking management, car wash point administration, customer inquiry handling, and an admin dashboard with real-time statistics.

## Features

- **User Authentication**: Secure login and registration system
- **Booking Management**: Full CRUD operations for service bookings
- **Car Wash Point Management**: Manage multiple service locations
- **Customer Inquiry Handling**: Track and respond to customer inquiries
- **Admin Dashboard**: Comprehensive statistics and analytics

## Tech Stack

| Component | Technology |
|-----------|-----------|
| **Backend** | PHP 8.0 |
| **Database** | MySQL |
| **Frontend** | HTML5, CSS3, Bootstrap 5 |
| **Server** | Apache / Nginx |
